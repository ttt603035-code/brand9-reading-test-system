# brand9-reading-test-system
An IELTS Reading test system for JSON import, testing, review, and local practice.

A lightweight single-file IELTS Reading test system built with HTML.

## Features

- Import IELTS Reading papers in JSON format
- Take reading tests locally
- Submit answers and review mistakes
- Track vocabulary and notes locally

## Copyright Notice

This repository only contains the test system code.
It does not include Cambridge IELTS PDFs, original passages, official questions, or copyrighted materials.

## How to Use

1. Download this repository
2. Open `index.html` in your browser
3. Import your own JSON paper
4. Start practicing
